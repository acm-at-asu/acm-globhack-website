# ASU-ACM-Website

Prototype for a new and improved ACM at ASU front page, using next.js, react, tsx

Some buttons link to old website asu.acm.org/ as it has more data on it, (also considering this is just a prototype for now some things still might be broken but most of it is just an easy fix so you dont have to worry about it too much for now.)

Vercel hosting is Work in Progress, currently being hosted on Aman's computer but you can host it on your own computer if you pull the repo. Add changes wherever you see fit or contribute.
